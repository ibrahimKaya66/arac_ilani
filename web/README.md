# Araç İlan – Web (Next.js)

Frontend uygulaması. Ana proje için [kök README](../README.md) dosyasına bakın.

## Geliştirme

```bash
npm install
npm run dev
```

`http://localhost:3000` adresinde çalışır.

## Ortam Değişkenleri

`.env.local` oluşturun:

```
NEXT_PUBLIC_API_URL=http://localhost:5094
```
